Pourquoi cette version statique

GitHub Pages ne lance pas PHP. Cette publication sert donc de copie figée du site pour la partie publique.

Rapide à charger

Pas de traitement serveur, pas de base de données, pas de dépendances backend. Le site s’ouvre immédiatement.

Lisible sur mobile

La mise en page est responsive pour garder un rendu propre sur téléphone, tablette et desktop.
